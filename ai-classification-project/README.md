
# 🌸 AI Flower Classification System

A beginner-friendly Machine Learning classification project built using Python and Scikit-learn. This project demonstrates the complete supervised learning pipeline including dataset loading, preprocessing, model training, prediction, and evaluation.

---

## 🚀 Project Overview

This AI system classifies Iris flower species using machine learning algorithms. The model is trained on flower measurements and predicts the flower category with high accuracy.

The project was built as part of an Artificial Intelligence training program focused on supervised learning fundamentals.

---

## ✨ Features

- 📊 Dataset loading and preprocessing
- 🧠 Supervised Machine Learning model
- 🌱 Iris flower classification
- 🔀 Train-test data splitting
- 📈 Accuracy evaluation
- 📉 Data visualization
- 🌐 Simple AI web app using Streamlit
